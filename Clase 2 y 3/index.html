Ir a Google

Que pasa cuando queremos hacer un hipervinculo con otro documento html que hayamos creado?. Vamos a hacer lo siguiente:

  • Abrimos etiqueta "a href="..." "" pero vamos a colocar dentro de las comillas la direccion del documento dentro de nustros archivos html
  • Para facilitar la busqueda del archivo, apretamos "ctrl+barra espaciadora" y nos salen los archivos recientes
  • Seleccionamos el archivo
  • No nos tenemos que olvidar de cerrar la etiqueta "/a"

Como resultado tenemos un hipervinculo como el siguiente:

Seguimos con las etiquetas: hay un tipo de etiquetas que permiten hacer listas ordenadas. Para hacer eso escribimos:

  1. La etiqueta "ol" que significa "order list": el browser me ordena una lista de forma numerada
  2. Dentro de esta etiqueta incluimos "li" que sigfica "list item". Tambien es una etiqueta de contenido asi que hay que recordar cerrarla.
NOTA:Para hacer esta lista se usaron estas etiquetas :)

Que pasa si queremos hacer una lista pero esta vez NO numerada?
Utilizamos:

  • La etiqueta "ul" que es una etiqueta de contenido.
  • Dentro de esta etiqueta ponemos: "li" (list item).
Y listo! tenemos una lista que se muestra con puntitos :)

Hay tambien etiquetas basicas que son de estilo. IMPORTANTE es recordar que el browser asigna ciertos estilos por defecto. Ya las estuve utlizando a lo largo del documento, pero vamos a repasar:

  • "b" nos pone un texto en negrita
  • "i"nos pone un texto en italica.
  • "mark"para remarcar.
  • "ins"subrayado
  • "small"texto mas chiquito.
  • "del" tachado
  • sup" para superindice
  • "strong" para indicar al browser que es una palabra importante. Se marca en negrita, pero la importancia de la etiqueta esta principalmente en que estas palabras se van a ubicar como de importancia en las busquedas de un motor de busquedas reacionadas.
  • "sub" para un subindice que puede tener un hipervinculo:
    about.index
