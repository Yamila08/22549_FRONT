Y cómo vuelvo al documento original ahora?
Hacemos un hipervinculo nuevo!, como el que voy a mostrar a continuacion:

Este es el link al doc "index.html" original

Como se hace?. Se siguen estos pasos:

  1. Abrimos la etiqueta "a"
  2. Ponemos de atributo "a href= " link del documento", por ejemplo en este caso "index.html", que en este caso se puede facilitar apretando el atajo de teclado "ctrl+barra espaciadora.
  3. No nos olvidamos de cerrar la etiqueta de "/a"
  4. Guardamos los cambios con "ctrl+s" para que el browser nos muestre los cambios en el documento

Y...listo!. Esto se hace para no volver al documento usando la flecha de "Atras" del browser.
