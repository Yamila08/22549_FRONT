Formularios

Cómo hacemos un fromulario?
Vamos a utilizar la etiqueta "form":

  1. "form" es una etiqueta contenedora, asi que la abrimos
  2. Dentro de "form" escribimos otra etiqueta:" label" (tambien es contenedora) y escribimos por ejemplo "Nombre".Cerramos "label".
  3. Escribimos una nueva etiqueta llamada "input" que nos permite abrir un cuadro.
  4. Dentro de "input" es necesario agregar Atributos como por ejemplo el atributo " type="text" " que nos permite escribir texto en el recuadro, por ejemplo vamos a llenarlo con el nombre "Light" (es el formilario para la inscripcion a la uni de Light Yagami).
  5. Podems tambien configurar la cantidad de items que pueden escribirse dentro con el atributo "maxlength="(una cifra x) 6" ", que quiere decir que podemos escribir 6 caracteres.


Para agregar dentro del formulario el cuadro para la contraseña, escribimos:

  • "label" Password "/label" (con esto describimos al usuario qué hacer en ese recuadro).
  • "input type="password", al que indicamos que la contraseña salga en puntitos.
  • Agregamos despues la etiqueta "input type="submit más el atributo "value="Enviar"para tener un boton de Envio.

Otro tipo de formulario: Check box"

Podemos hacer un tipo de formulario tipo "check box":

  • Decidimos el tema del formulario y lo escribimos. Por ejemplo: "Cuales son tus hobbies?"
  • Usamos "label"+un texto x + "/label", por ejemplo "musica"
  • Usamos "input type="check box" value="musica"
  • Podemos repetir lo mismo para otras opciones como "series","pelis",etc.
Hacemos un ejemplo:
Cuáles son tus hobbies?

Tambien podemos hacer un Check box donde solo se pueda seleccionar UNA de las opciones.

  • Elegimos un tema: "Sos un fantico de GOT?"
  • "label"+ la opcion, este caso "si" + "/label"
  • "input type="radio" (este es un nuevo atributo), value="si", name="opciones(el name tiene que ser el mismo en las dos opciones para que solo sea posible seleccionar UNA opcion, equivalente a "o" (si o no)".
  • Repetimos estos pasos para hacer la segunda opcion con "value="no".
